# Professional Site

Sito web professionale personale sviluppato per presentare profilo, competenze, esperienze e progetti.

Il progetto ha l'obiettivo di fornire una presenza online semplice e professionale, facilmente deployabile e modificabile.

## 📌 Descrizione

**Professional Site** è un sito web statico pensato come portfolio personale.
Permette di mostrare informazioni professionali come:

* Presentazione personale
* Competenze tecniche
* Esperienze lavorative
* Progetti sviluppati
* Contatti

Il sito può essere pubblicato facilmente online tramite hosting statico o tramite **GitHub Pages**.

## 🚀 Funzionalità

* Layout semplice e professionale
* Design responsive
* Struttura facilmente modificabile
* Codice pulito e organizzato
* Deploy semplice

## 🛠️ Tecnologie utilizzate

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## 📂 Struttura del progetto

```
professional-site
```

## ⚙️ Installazione

Clonare il repository:

```bash
git clone https://github.com/msabetta/professional-site.git
```

Entrare nella cartella del progetto:

```bash
cd professional-site
```
Eseguire lo script run.py

```python
python run.py
```

## 🌐 Deploy

Il sito può essere pubblicato tramite:

* GitHub Pages
* Hosting statico
* Server web tradizionale

Per pubblicarlo con GitHub Pages:

1. Aprire le **Settings** del repository
2. Andare nella sezione **Pages**
3. Selezionare il branch `main`
4. Salvare

Il sito sarà disponibile all'indirizzo:

```
https://msabetta.github.io/professional-site
```
